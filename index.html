Projets

Trottiprêt. Ce projet consiste à créer une plateforme de prêt de trottinette entre étudiants en utilisant la méthode agile. Chaque étudiant peut prêter ou emprunter une trottinette à un autre étudiant pour un lieu et une période donnée. Projet developpé en C++ avec QT, utilisation de doxygen pour générer la documentation, gestion d'une base de données en local. (Projet en L3)

Programmation avancée. Ce projet a été élaboré en C avec la librairie SDL tout en assurant une bonne allocation dynamique pour éviter les fuites de mémoires. C'est un petit jeu en 2D où l'on peut se déplacer dans un labyrinthe avec un tank et tirer. Le but : atteindre les drapeaux pour passer aux niveaux suivants sans se faire toucher par les tourelles ! (Projet en L2)

Projet web. Site web de recettes de boissons. Utilisation d'une base de donnée, JavaScript, Ajax, PHP, HTML et CSS. (Projet L2)

Procrast : site web où les utilisateurs peuvent se créer un profil et se connecter, créer des listes de tâches, ajouter des tâches, inviter des collaborateurs sur leurs listes, etc. Utilisation de PHP, JavaScript, HTML, CSS, SQLite... (Projet L3)

PacWomen. Sur la page d'acceuil, nous pouvons soit commencer la partie, quitter ou bien aller dans le menu option pour choisir l'un des deux personnages à incarner.On peut également choisir le niveau de difficulté que l'on veut pour la partie. Le but : passer les niveaux sans se faire tuer par les monstres (implémentés avec l'algorithme A*). Utilisation de Java et Maven. (Projet M1)

Développement d'une application "covid" permettant à un utilisateur de renseigner des amis ainsi que des lieux qu'on a visité, et de notifier toutes les personnes susceptibles d'être contaminées si l'on se déclare soi-même positif (amis et personnes présentent au même endroit au même moment). Utilisation de J2EE avec Eclipse, SQL, Bootstrap. (Projet M1)

Petit projet de web scraping avec Python et Beautiful Soup permettant de récuperer certaines informations à partir de l'url d'entré.

Puissance 4. Programme implémanté en C pour jouer au puissance 4 contre l’ordinateur. L’ordinateur choisit ses coups en appliquant l’algorithme de recherche d’arbre Monte Carlo UCT en temps limité.
